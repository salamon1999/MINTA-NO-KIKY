# Minta Nomor Telpon Web App 💖

A fun and interactive web application to ask for a "crush's" phone number.

## Features
- **"Tdk" Button**: Dodges any attempt to click it!
- **"Iya" Button**: Transitions to a sweet phone number input screen.
- **Encouragement Message**: A heartwarming message at the end.
- **Responsive Design**: Works on mobile and desktop.

## How to Run Locally

1.  Navigate to the project directory:
    ```bash
    cd minta-nomor
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
4.  Open your browser at the URL shown in the terminal (usually `http://localhost:5173`).
